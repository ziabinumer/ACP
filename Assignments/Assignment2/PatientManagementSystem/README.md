# Patient Management System

## Project Structure
This is a Java Swing-based Patient Management System with role-based access control.

## Setup
1. Configure database connection in `resources/config/database.properties`
2. Run `Main.java` to start the application
3. Default login credentials will be set up on first run

## Features
- Administrator: Full CRUD operations on patients, doctors, and diseases
- Guest: Read-only access with limited search capabilities

## Technologies
- Java Swing (GUI)
- JDBC (Database connectivity)
- MySQL/SQLite (Database)
